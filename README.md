# claude-playground

Claude と GitHub MCP の連携を試すための実験リポジトリです。

## 構成

```
claude-playground/
├── aarm/                  # AARM プロトタイプ
│   ├── sdk/               # AARM SDK パッケージ (pip install -e で使う)
│   ├── agent/             # エージェントとデモ
│   └── README.md
├── mermaid_samples/       # Mermaid 図サンプル集
└── web_research_agent/    # Web リサーチエージェントサンプル
```

## 環境

- Claude Desktop
- GitHub MCP Server (`@modelcontextprotocol/server-github`)
